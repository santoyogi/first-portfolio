The CSS Box Model

The CSS box Model has four parts that can all be modified: the content, padding, border and margin. The content holds text or image while the padding is the area around the content. Then the border is simply the border that goes around the content as well as the padding. Finally, Margin is the area that is cleared around the border.

Positioning

Positioning has to do with the type of method used to position an element. Below are 3 positioning examples.

Relative Position

Using relative positioning will move the element relative to its regular position, leaving a gap where it once was.

Using relative positioning to move this text from the left 100px and from the top 15px

Absolute Position

Using absolute positioning will move the element relative to the document body since it is embedded in the body section of the html file. However the position can be relative to whatever you associate the element to. In this case, it is relative to this section of absolute postioning.

This is an element with absolute positioning. Relative to this section it is positioned 100px from the top and 100 px from the left





Fixed Position

Fixed positioning will postion an element relative to the viewport. This means that it will stay in the same place even when you scroll up or down. The element is red and in the top right corner of the screen.
